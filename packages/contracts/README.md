# packages/contracts

Gemeinsame Verträge und Schemas für SignalOS.

Geplante Inhalte:

- API-Typen
- Agenten-Output-Schemas
- Event-Payloads
- Zustands- und Rollenmodelle

Dieses Paket ist zentral für Konsistenz, Nachvollziehbarkeit und sichere Übergaben zwischen UI, API, Agenten- und Workflow-Layer.
